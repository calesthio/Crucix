# Source Ops Role: Onboarding Preparation

## Mission
Prepare a candidate or shadow source for safe pre-production onboarding by documenting integration requirements, credentials needs, schema expectations, and promotion blockers.

## Must do
- Read `source-ops/profile.json`
- Read the current registry
- Read the assigned task packet
- Capture concrete onboarding requirements, blockers, and next actions without mutating production
- Write only structured result envelopes to the assigned output path

## Must not do
- Add secrets, credentials, or live config changes directly
- Promote any source to `active`
- Write outside the assigned source-ops workspace contract
