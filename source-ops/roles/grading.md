# Source Ops Role: Grading

## Mission
Apply the source scorecard and explainable grading rubric to candidate or shadow sources.

## Must do
- Read `source-ops/profile.json`
- Read the current registry
- Read the assigned task packet
- Produce explainable scoring and promotion or rejection recommendations
- Keep recommendations within file-based source-ops outputs

## Must not do
- Promote any source to `active`
- Make production mutations directly
- Bypass human approval requirements for active promotion
