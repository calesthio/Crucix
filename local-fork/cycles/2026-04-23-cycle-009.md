## Results

- Added stable signal ID generation in `server.mjs` via:
  - `signalId()`
  - `attachSignalIds()`
- Compact brief responses now include stable IDs for:
  - `topCorroborated`
  - `topSuspect`
  - top corroborated list entries
  - top suspect list entries
- Compact text now includes the top-item IDs inline, for example:
  - `Top corroborated [corroborated-regional-corroboration-middle-east-0]: ...`
- Drill-down endpoint now supports `id=` selection in addition to `index=`.
- This makes the iMessage layer routing-friendly and less brittle.
