# Crucix Cycle 056 - 2026-04-24

## Summary
- Completed Epic 17 effort 17.1 by hardening macro-card rendering so missing, failed, or non-finite macro values no longer leak operator-hostile strings such as `NaN`, `NaNT`, or malformed percent displays.

## What changed
- Updated `dashboard/public/jarvis.html`.
- Added shared finite-number formatting helpers in the macro/markets panel render path:
  - `finiteNumber()`
  - `fmtFixed()`
  - `fmtSignedPct()`
  - `fmtCompactPrice()`
- Reworked macro and market card formatting so the panel now:
  - treats non-finite and absent values as unavailable
  - renders `--` placeholders instead of malformed numeric strings
  - preserves valid signed percent moves for healthy feeds
  - avoids leaking bad change text when market change percentages are missing
- Hardened the following macro card outputs specifically:
  - VIX
  - Fed Funds
  - GSCPI
  - CPI MoM
  - Unemployment
  - market quote cards used for indexes/crypto/rates

  - `NaN`
  - `NaNT`
  - `undefined`
  - `null`
  - malformed placeholder composites

  - `Fed Funds -> --`
