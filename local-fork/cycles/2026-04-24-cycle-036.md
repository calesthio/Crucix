## Classification model
Per source, the current sweep can now classify failures as:
- `setup-debt`
- `external-limit`
- `live-outage`
- `parse-failure`
- `other-failure`

Examples now separated in the live runtime:
- Missing API keys or credentials -> `setup-debt`
- OpenSky cooldown / provider throttling and BLS quota exhaustion -> `external-limit`
- Space/CelesTrak `fetch failed` -> `live-outage`
- malformed payload states -> `parse-failure`
