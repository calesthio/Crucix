# Crucix Dev Cycle 130 - Source-control audit history and undo metadata

- Date: 2026-04-26
- Plan item: `epic-33 / effort-33.5`
- Status: completed

## Goal
Add source-control audit history and undo metadata for suppress and quarantine actions so local-admin mutations are attributable, reviewable, and safely reversible during incident triage.

## Changes shipped
- Updated `server.mjs`
  - added source-control audit storage at `runs/source-control-audit.json`
  - added shared source-control mutation helper used by both source-console and review-workflow actions
  - added per-action before/after state capture, `changed` detection, and undo metadata for suppress/quarantine actions
  - added `GET /api/source-ops/audit`
  - extended `POST /api/source-ops/control` responses with audit and undo details
  - extended `POST /api/review-workflow/action` source actions to support:
    - `suppress-source`
    - `unsuppress-source`
    - `quarantine-source`
    - `clear-quarantine`
  - extended settings/source-console contract with:
    - `source-ops-control-v2`
    - audit endpoint metadata
    - recent source-control audit entries
- Updated tests:
  - `test/settings-contract.test.mjs`
  - `test/operator-settings-persistence.test.mjs`
  - `test/settings-endpoint-contract.test.mjs`

## Validation
- Tests passed:
  - `node --test test/settings-contract.test.mjs test/operator-settings-persistence.test.mjs test/settings-endpoint-contract.test.mjs`
- Runtime validation performed through booted-server endpoint tests, including verification that:
  - `/api/source-ops/audit` returns `version = "source-control-audit-v1"`
  - source-action responses include undo metadata such as `unsuppress-source` and `clear-quarantine`
  - review-workflow source actions can be reversed with `unsuppress-source`

## Backlog
- No new backlog items were identified in this cycle.

## Notes
- Undo metadata is intentionally explicit and endpoint-scoped so incident triage can reverse a recent suppress or quarantine action without reconstructing the prior control state by hand.
