## Counter model
Per source, the current sweep can now register:
- `fallbackCount`
- `parseFailureCount`
- `llmFallbackCount`
- `emptyResultCount`

Examples captured by the classifier:
- OpenSky cache-served or fallback query modes count as fallback.
- Telegram `web_scrape` counts as fallback.
- ReliefWeb HDX fallback counts as fallback.
- News clustering heuristic fallback pressure counts against `GDELT` as an LLM-fallback event.
- Empty structured results like WHO with no outbreak items can count as empty results when no explicit error is present.
